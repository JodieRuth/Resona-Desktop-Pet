# 🖼️ 用户界面组件 (UI Components)

UI 模块位于 `resona_desktop_pet/ui/`，采用 PySide6 构建。

## 1. MainWindow (`luna/main_window.py`)
整个程序的核心 UI 控制器。
- **无边框透明窗口**：实现了桌面宠物的透明悬浮效果，支持无边框、无阴影、工具窗口样式。
- **状态管理**：管理宠物的"闲置"、"思考"、"说话"、"录音"等各种状态，并协调各组件的显示。
- **交互处理**：处理鼠标拖拽、缩放（Alt+滚轮）、右键菜单等交互。
- **自动隐藏/淡出**：支持全屏模式自动隐藏，以及长时间不操作时的闲置淡出效果。
- **物理引擎集成**：与物理引擎桥接，支持拖拽抛掷、碰撞反弹等物理效果。
- **文件拖入支持**：支持检测文件拖入事件（开发中）。
- **置顶控制**：支持定时强化窗口置顶，确保宠物始终可见。

## 2. CharacterView (`luna/character_view.py`)
专门负责立绘显示。
- **多服装支持 (Outfits)**：根据 `sum.json` 索引不同的服装目录，支持运行时切换。
- **表情切换**：实现从情感标签到具体图片文件的随机或确定性选取。
- **高性能渲染**：优化了缩放渲染，保证在透明背景下的平滑显示。
- **鼠标交互**：支持鼠标悬停、点击、拖拽等交互状态检测。
- **动态缩放**：根据配置自动计算初始缩放比例，支持 Alt+滚轮手动缩放。

## 3. IOOverlay (`luna/io_overlay.py`)
对话框组件。
- **输入/输出一体化**：支持用户文本输入和 LLM 回复显示。
- **自动排版**：根据立绘位置自动调整对话框的弹出位置（左上或右上）。
- **动画效果**：具备渐变显示效果，提升视觉体验。
- **角色名称显示**：显示当前角色名称和用户名。
- **重新生成按钮**：支持点击重新生成 LLM 回复。

## 4. 托盘图标 (`tray_icon.py`)
- 提供后台运行支持。
- 快捷操作菜单（打开设置、切换角色包、退出等）。
- 服装切换菜单：支持快速切换当前角色的不同服装。
- 资源包切换菜单：支持快速切换不同角色资源包。

## 5. 设置对话框 (`settings_dialog.py`)
- **全量设置**：提供详细的 API 配置、TTS 参数、UI 行为、物理引擎等设置。
- **多标签页**：按功能分类组织设置项（通用、LLM、STT、SoVITS、物理、行为等）。
- **实时保存**：修改配置后可立即保存生效。
- **模型选择**：支持选择不同的 LLM 提供商和模型。

## 6. 调试面板 (`debug_panel.py`)
开发者工具，用于测试和调试。
- **手动触发响应**：可手动指定情感标签、显示文本、TTS 文本，模拟 LLM 返回。
- **情感选择**：下拉选择可用的情感标签。
- **TTS 语言切换**：支持切换语音合成语言。
- **快捷操作**：方便开发者测试特定场景下的宠物表现。
